Scoring
composition.

Score = mensuel actuel + k × tendance de la saison (plafonnée à ±150). Le coefficient k=0,6 par défaut est calibré sur l'historique (corrélation 0,64, optimum de prédiction). Ajuste-le avec le curseur pour tester d'autres pondérations.

Classement scoring
0.6
0 = classement mensuel pur · 0,6 = optimal (données) · >0,7 = sur-pondéré
#JoueurMensuelTendance saisonScoreTraj.

Mensuel = classement mensuel FFTT actuel. Tendance = évolution sur 2025/26. Trajectoire : ↗ progression (≥+30), ↘ baisse (≤−30). Données API FFTT. Page non listée (accès direct uniquement).
