Scoring
composition.

Score = mensuel actuel + k × tendance (plafonnée à ±250). La tendance est la pente de régression linéaire du classement officiel sur les 4 dernières phases (≈ progression sur 2 ans, lissée). Le poids k=0,6 par défaut est ajustable au curseur.

Classement scoring
0.6
0 = mensuel pur · 0,6 = équilibré · >0,7 = forme sur-pondérée
#JoueurMensuelTendance 4 phasesScoreTraj.

Mensuel = classement mensuel FFTT actuel. Tendance = pente de régression sur les 4 dernières phases du classement officiel (survole pour voir le détail). Trajectoire : ↗ progression (≥+30), ↘ baisse (≤−30). Données API FFTT. Page non listée (accès direct uniquement).
